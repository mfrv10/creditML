# Data Sources

## Public Credit Datasets

### 1. UCI German Credit Data
- **Source**: [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/datasets/statlog+(german+credit+data))
- **Description**: 1000 observations with 20 attributes for credit risk assessment
- **Features**: Credit history, purpose, employment, personal status, etc.

### 2. Lending Club Loan Data
- **Source**: [Kaggle](https://www.kaggle.com/datasets/wordsforthewise/lending-club)
- **Description**: Historical loan data from Lending Club platform
- **Features**: Loan amount, interest rate, employment length, annual income, etc.

### 3. Give Me Some Credit
- **Source**: [Kaggle Competition](https://www.kaggle.com/c/GiveMeSomeCredit)
- **Description**: 150,000 borrowers with financial history
- **Features**: Age, debt ratio, monthly income, number of dependents, etc.

## Synthetic Data

Generated synthetic datasets are stored in `sample_data/` directory for testing and demonstration purposes.
